# AstroBookings Briefing

AstroBookings is a training product for a fictional space travel booking.
It helps learn software development during hands-on exercises and workshops.

The project will be an API for managing space travel bookings.

Users can manage rockets with a range and passenger capacity.
They can browse available vehicles and seats before planning trips.

Users can schedule launches with price, minimum passenger for profit.
Launches can be confirmed, cancelled, or rescheduled as needed.
They can review launch lists to track upcoming travel options.

Users can register customers and keep traveler profiles up to date.
Users can create bookings that connect travelers to launches.
They can manage booking lifecycle from creation to cancellation.

Payments are processed immediately for bookings.
In case of cancellation, refunds are issued based on the time of cancellation.

No database is needed, as data is stored in memory for simplicity.
No security features are implemented, as the focus is on learning core concepts.
Payments gateway is simulated with a simple fake that always succeeds.
